# 도서 대여 및 재고 관리 시스템

## 프로젝트 소개
도서 정보를 등록하고, 대여 및 반납 상태를 관리하며, 현재 재고를 확인할 수 있는 JSP/Servlet 기반 웹 프로젝트입니다.

## 기술 스택
- Java
- JSP / Servlet
- JDBC
- MariaDB 또는 MySQL
- HTML / CSS / JavaScript
- Apache Tomcat
- Git & GitHub

## 주요 기능
- 도서 등록
- 도서 목록 조회
- 도서 수정
- 도서 삭제
- 도서 대여
- 도서 반납
- 대여 상태 확인
